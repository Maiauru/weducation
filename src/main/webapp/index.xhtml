Web Education (weducation)

Приветствуем вас в тестовом приложении Weducation!

Данный ресурс создан для того, чтобы вы могли ознакамливаться с функционалом по мере разработки приложения и своевременно высказывать свою конструктивную критику и пожелания.

На данный момент в финальной стадии находится разработка хранения учебных планов по специальностям.
